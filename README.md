# miniProjects

A collection of mini projects including HTML, CSS, Python projects, and Advent of Code 2025 solutions.

## How to Clone This Repository

### Using HTTPS (Recommended)

To clone this repository using HTTPS, run the following command in your terminal:

```bash
git clone https://github.com/schorsch7198/miniProjects.git
```

### Using SSH

If you have SSH keys set up with GitHub, you can clone using SSH:

```bash
git clone git@github.com:schorsch7198/miniProjects.git
```

### Using GitHub CLI

If you have GitHub CLI installed, you can also clone with:

```bash
gh repo clone schorsch7198/miniProjects
```

## After Cloning

Once cloned, navigate into the project directory:

```bash
cd miniProjects
```

## Project Structure

The repository contains the following directories:

- **AoC25/** - Advent of Code 2025 solutions
- **CSS/** - CSS projects and examples
- **HTML/** - HTML projects and tasks
- **first_Projects/** - Initial projects including various Python programs

## Getting Started

Explore the different directories to see various projects. Each subdirectory may contain its own specific instructions or examples.

## Requirements

Different projects may have different requirements. Check individual project directories for specific dependencies.
